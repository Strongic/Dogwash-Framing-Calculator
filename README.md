# Dogwash Framing Calculator
I've always wanted a simple calculator to show how much wood (and at what length) I would need to build dogwash crates. Each crate needs a pallet and 2 walls. This lightweight site does exactly this, it generates the necessary materials needed to build dogwash crates in both the DW2 and Mini variety.

Still a WIP but the basic logic is working, check it out:
https://strongic.github.io/Dogwash-Framing-Calculator/

running locally:
python3 -m http.server [port]

ctrl + c to kill
fg to pull into forground and then kill
